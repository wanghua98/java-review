## POST 用户登录

POST /api/login

> Body 请求参数

```json
{
    "username": "admin",
    "password": "aa"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|object| 是 |none|

> 返回示例

> 200 Response

```json
{"code":400,"message":"账号密码错误","data":null}
```

## POST 上传分片

POST /api/file/upload

> Body 请求参数

```yaml
file: file:///Users/dao/Desktop/sintinnel.mp4

```

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|file|query|string| 否 |none|
|taskId|query|string| 否 |none|
